Favicon Setup Instructions:

  1. The logo-only.png is already set as favicon.png
  2. For better browser support, you can convert logo-only.png to favicon.ico
  3. Use online tools like favicon.io or realfavicongenerator.net
  4. Upload logo-only.png and download the generated favicon.ico
  5. Replace the current favicon.png with the generated favicon.ico
